Cheat Sheet

CSS

Border

Defines the border around an element, including its width, style, and color.

Example

border: 1px solid black;

Borders: top, bottom, left, right

Specifies different borders for each side of an element (top, right, bottom, and left).

Example

border-top: 1px solid black;
border-right: 2px dotted red;

Border-radius

Defines the radius of the element's corners, allowing for rounded edges.

Example

border-radius: 10px;

Background-image

Sets an image as the background of an element.

Example

background-image: url('image.jpg');

Color

Specifies the color of text.

Example

color: blue;

Font

Shorthand property for setting font-style, font-variant, font-weight, font-size, line-height, and font-family.

Example

font: italic small-caps bold 16px/1.5 Arial, sans-serif;

Font-Family

Specifies the font family for text.

Example

font-family: 'Times New Roman', Times, serif;

Font-style, Font-weight

Defines the style (e.g., italic) and weight (e.g., bold) of the font.

Example

font-style: italic;
font-weight: bold;

Width, Height

Sets the width and height of an element.

Example

width: 100px;
height: 200px;

List-style

Defines the style of list-item markers (e.g., disc, circle, square).

Example

list-style: square inside;

Margin

Sets the margin area around an element.

Example

margin: 20px;

Padding

Sets the padding area inside an element, between the content and the border.

Example

padding: 10px 20px;

Overflow

Specifies what happens if content overflows an element's box.

Example

overflow: scroll;

Display

Specifies how an element is displayed (e.g., block, inline, none).

Example

display: inline-block;

Position

Defines the positioning method for an element (e.g., static, relative, absolute, fixed, sticky).

Example

position: absolute;
top: 50px;
left: 100px;

Filter

Applies graphical effects like blur or color shift to an element.

Example

filter: blur(5px);
